Welcome to the official repository for the **Five-day Hands-on Workshop Program on Bioinformatics for One Health and Pathogen-Specific Research**. 

This repository contains all the computational notebooks, scripts, and datasets required for the hands-on sessions. 

**Organised by:** CSIR-Institute of Genomics and Integrative Biology, New Delhi 
**Funded by:** Indian Council of Medical Research (ICMR)  

---

## 📁 Repository Structure

The curriculum is divided into daily folders. Each folder contains the Jupyter/Colab Notebooks (`.ipynb`) and a `data/` subdirectory for that day's specific exercises.

* `DAY_1/` - Linux Basics & Environment Setup
* `DAY_2/` - Genome Assembly Workflow & Quality Assessment
* `DAY_3/` - Comparative Genomics, Variant Calling, & Functional Enrichment
* `DAY_4/` - Antimicrobial Resistance (AMR) & Microbiome Analysis
* `DAY_5/` - Isoform Analysis & Graph-Augmented AI/ML 

---

## 💻 Prerequisites & Setup

All hands-on sessions will utilize preconfigured HPC/Cloud-based platforms for reproducibility. However, participants are expected to bring their own laptops for the practical training sessions. 

**System Requirements:**
* A modern web browser (Chrome/Firefox/Safari) for accessing Google Colab.
* A stable internet connection.
* *Note for macOS users:* Your MacBook's native Terminal runs Zsh by default; we will be utilizing standard Bash environments in our Colab/HPC instances, so keep standard Linux syntax in mind!

Workshop materials and sample datasets will be provided within each daily folder.

## 📅 Daily Schedule Overview

### [Day 1: The Genomic Compass](DAY_1/)
*Navigating One Health Bioinformatics* 
* **Morning:** Inauguration, Talk on Harnessing Genomic Technologies.
* **Hands-on 1A & 1B:** Linux Basics for Bioinformatics, Command-line navigation, core Unix utilities, BASH scripting, and Conda package management.
* **Evening:** CSIR-IGIB Bio-bank & HPC Facility visit.

### [Day 2: The Assembly Quest](DAY_2/)
*From Raw Data to Complete Genomes*
* **Morning:** Talks on Pathogenomics, AI-enabled insights, and Principles of genome assembly.
* **Hands-on 2A:** Quality assessment and visualization (FastQC, Fastp, MultiQC).
* **Hands-on 2B:** Viral Genome Assembly (Setu pipeline, Spades) and Completeness Check (QUAST/BUSCO).

### [Day 3: Comparative Genomic Analysis](DAY_3/)
*Deciphering Genetic Relationships* 
* **Morning:** Understanding Pathogen Diversity.
* **Hands-on 3A:** SNP & Variant Calling, Mapping Sequencing Reads.
* **Hands-on 3B:** Sequence Alignment, Phylogenetic tree Construction.
* **Spotlight Tools:** Pathway enrichment analysis with getEnrich and Chromosomal rearrangement with Chitra.
* **Hands-on 3C:** croSSRoad pipeline for SSR-Driven Comparative Genomics.

### [Day 4: AMR Surveillance](DAY_4/)
*Genomic Approaches for Antimicrobial Resistance*
* **Morning:** ARKbase: Antimicrobial Resistance Knowledgebase 1.0
* **Hands-on 4A & 4B:** Exploring AMR databases, genome-based resistance tools, and analytical workflows.
* **Hands-on 4C:** Microbiome Analysis and Marker Feature Identification using MarkerML.

### [Day 5: Translating Genomic Insights](DAY_5/)
*From Genomes to Public Health* [cite: 212]
* **Morning:** Talks on Alternate Transcript Isoforms and Graph-Augmented AI/ML for AMR.
* **Afternoon:** Valedictory & Certificate Distribution, followed by Delhi Darshan.
